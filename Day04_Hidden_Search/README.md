# Hidden Search - Expandable Interface

A simple expandable search interface that transforms from a button into a full search input with smooth animations.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Transitions, Transforms), JavaScript  
**Focus:** CSS animations, state management  


## Features
- Expandable search input that grows from 50px to 200px width
- Smooth button translation that follows the expanding input
- Toggle between collapsed and expanded states
