# Drag & Drop Interface

A simple, interactive drag and drop component built with JavaScript.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Transitions), JavaScript  
**Focus:** HTML5 Drag & Drop API, DOM manipulation, event handling  


## Features
- Drag and drop image between containers
- Visual feedback during drag operations
- Mobile-responsive design
- Smooth animations and transitions


## Improvements Made
- **Enhanced visual feedback** – Added smooth transitions, hover effects, and better color scheme
- **Mobile responsiveness** – Optimized sizing and layout for smaller screens
- **Code quality** – Cleaner event listener setup


## What I Learned
- HTML5 Drag & Drop API with dragstart, dragend, dragover, dragenter, dragleave, and drop events
- DOM manipulation for moving elements between containers dynamically
- CSS transitions for smooth visual feedback
