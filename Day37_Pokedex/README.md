# Pokedex - Pokemon Information Cards

A dynamic Pokemon showcase displaying the first 151 Pokemon with their stats, types, and information fetched from the PokeAPI, featuring card-based design and interactive hover effects.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Grid, Transitions), JavaScript (Fetch API)  
**Focus:** API integration, dynamic content generation, grid layouts  


## Features
- Fetches and displays data for the first 151 Pokemon from PokeAPI
- Card-based layout with Pokemon images, names, IDs, and types
- Dynamic background colors based on Pokemon primary type
- Stats display showing HP, Attack, Defense, Speed, and more
- Responsive grid layout that adapts to different screen sizes
- Automatic sequential data loading for all Pokemon


## Improvements Made
- **Hover shine effect** – Added animated shine overlay on card hover for visual appeal
- **Card-like design** – Added card styling with shadows and rounded corners
- **Enhanced visual polish** – Improved overall aesthetics with better spacing and transitions


## What I Learned
- Fetch API for making HTTP requests to external APIs
- Async/await patterns for handling asynchronous operations
- Dynamic DOM generation with template literals
- CSS Grid for responsive card layouts
- Type-based conditional styling with color mapping
- Data transformation and formatting from API responses
