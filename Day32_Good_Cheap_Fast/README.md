# Good, Cheap, Fast - Interactive Project Triangle

An interactive demonstration of the classic project management triangle where users can select any two of three options (Good, Cheap, Fast) with automatic constraint enforcement and smooth toggle animations.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Custom Properties, Animations), JavaScript  
**Focus:** State management, event handling, constraint logic  


## Features
- Interactive toggle switches demonstrating the project management triangle principle
- Automatic constraint enforcement preventing all three options from being selected
- Smooth toggle animations with cubic-bezier easing for natural movement
- Clear visual feedback showing which options are currently active


## Improvements Made
- **Enhanced visual design** – Implemented dark theme with custom CSS variables and professional color palette
- **Improved toggle animation** – Added smooth cubic-bezier transitions with bounce effects
- **User experience improvement** – Added descriptive subtitle text to explain the constraint rule clearly


## What I Learned
- Event handling optimization with forEach loops and addEventListener for multiple elements
- State management tracking multiple checkbox states with automatic deselection logic
- CSS custom animations with cubic-bezier timing functions for natural movement
