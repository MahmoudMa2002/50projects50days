# Netflix Mobile Navigation - Animated Menu Interface

A Netflix-inspired mobile navigation menu with smooth slide-in animations, backdrop overlay, and enhanced user experience featuring clickable background dismiss functionality.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Transforms, Transitions), JavaScript  
**Focus:** Mobile navigation patterns, overlay effects, UX design  


## Features
- Slide-in navigation menu with smooth transform animations
- Dark backdrop overlay that appears when menu opens
- Netflix-branded styling with logo and color scheme
- Click outside to close menu functionality
- Interactive list items with hover effects
- Toggle between open and closed states with hamburger icon


## Improvements Made
- **Hover effects** – Added interactive hover states for navigation items
- **Darkened backdrop** – Added overlay that dims background when menu is active
- **UX improvements** – Added click-outside-to-close functionality for better usability


## What I Learned
- Mobile navigation menu patterns and animations
- Backdrop overlay implementation with click handlers
- Transform-based slide animations
- Event propagation and click-outside detection
- Toggle state management for navigation
- Creating branded UI components
