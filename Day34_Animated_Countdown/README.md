# Animated Countdown - Dynamic Sequence Animation

A countdown timer with smooth rotation animations and event-driven sequencing, featuring automatic progression through numbers with a finale presentation and replay functionality.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Keyframes, Transforms), JavaScript  
**Focus:** Animation sequencing, event-driven programming  


## Features
- Smooth countdown animation sequence from 3 to 0 with automatic progression
- Dynamic rotation animations with natural bounce effects using keyframes
- Event-driven animation system that responds to animationend events
- Finale presentation with scaling animation and gradient text effects
- Replay functionality with complete DOM reset and animation restart capability


## Improvements Made
- **Better typography** – Enhanced text styling with gradient backgrounds and proper spacing
- **Better button design** – Styled replay button with hover effects and transitions


## What I Learned
- Animation event handling with animationend events
- CSS transform combinations (translate, rotate, scale)
- Sequential animation logic with automatic progression
- Event-driven programming for animation sequences
- DOM state management with dynamic class toggling
- CSS keyframe animations with rotation and easing
