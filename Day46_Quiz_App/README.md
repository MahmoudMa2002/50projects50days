# Quiz App - Interactive Trivia Game

An interactive quiz application that presents multiple-choice questions, tracks user answers, and displays final scores with the ability to reload and retake the quiz.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Flexbox), JavaScript  
**Focus:** State management, dynamic content, quiz logic  


## Features
- Multiple-choice quiz questions with radio button selection
- Dynamic question progression with submit functionality
- Score tracking and final results display
- Reload functionality to retake the quiz
- Clean card-based interface with modern styling
- Answer validation and result calculation
