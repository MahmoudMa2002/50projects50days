# Random Image Feed - Infinite Gallery

A dynamic image gallery that fetches random photos from Unsplash API, featuring infinite scroll functionality, load more button, and modern grid layout with hover effects.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Grid, Transitions), JavaScript (Fetch API)  
**Focus:** API integration, dynamic loading, infinite scroll  


## Features
- Fetches random images from Unsplash API
- Responsive grid layout that adapts to screen size
- Load more functionality to fetch additional images
- Smooth image loading with fade-in effects
- Hover effects on images for visual feedback
- Clean, modern gallery interface


## Improvements Made
- **Load more button** – Added manual control to fetch additional images on demand
- **Enhanced styling** – Applied custom fonts that fit the gallery theme
- **Hover effects** – Implemented smooth scale and overlay effects on image hover


## What I Learned
- Unsplash API integration for random images
- Dynamic image loading and DOM insertion
- Infinite scroll patterns with load more functionality
- CSS Grid for responsive gallery layouts
- Image optimization and lazy loading concepts
- Event handling for manual content loading
