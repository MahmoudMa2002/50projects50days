# Password Strength Indicator - Dynamic Background Blur

An interactive login interface where the background blur gradually clears as the user types a stronger password, featuring real-time strength calculation and visual feedback.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Filters, Transitions), JavaScript  
**Focus:** Password validation, dynamic blur effects, UX feedback  


## Features
- Dynamic background blur that reduces as password strength increases
- Real-time password strength calculation based on length and character variety
- Smooth blur transitions from heavily blurred to crystal clear
- Visual feedback showing password strength progression
- Responsive design with centered login card interface
- Clean, modern UI with form validation feedback


## Improvements Made
- **Strength indicator bar** – Added visual progress bar showing password strength levels
- **Enhanced calculation** – Improved password strength function with better criteria
- **Better UX feedback** – Implemented real-time visual indicators for user guidance


## What I Learned
- CSS filter properties for dynamic blur effects
- Password strength validation algorithms
- Real-time input event handling
- Mathematical mapping between password strength and blur values
- Creating visual feedback systems for user input
- Form UX best practices for password fields
