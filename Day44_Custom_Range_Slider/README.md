# Custom Range Slider - Interactive Value Control

A custom-designed range slider with draggable thumb, dynamic value display, and smooth visual feedback, featuring enhanced styling and improved interaction.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Custom Properties, Transitions), JavaScript  
**Focus:** Input customization, drag interactions, value tracking  


## Features
- Custom-styled range slider with branded appearance
- Draggable thumb control with smooth movement
- Dynamic value display that follows thumb position
- Real-time value updates as slider moves
- Visual feedback with color transitions
- Responsive design with proper touch support


## Improvements Made
- **Draggable functionality** – Made slider fully draggable with click-and-hold support
- **Polished styling** – Refined overall design with shadows, borders, and modern themes
- **Improved interactions** – Added smoother transitions and better touch handling


## What I Learned
- Custom styling for native input range elements
- Input event handling for real-time value updates
- Positioning labels dynamically based on slider value
- CSS custom properties for theme consistency
- Cross-browser range slider styling techniques
- Creating intuitive drag-and-drop interactions
