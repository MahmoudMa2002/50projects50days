# Testimonial Box Switcher - Rotating Reviews

An automated testimonial carousel that cycles through user reviews with smooth transitions, featuring a visual progress bar, manual navigation arrows, and modern card design.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Animations, Transitions), JavaScript  
**Focus:** Carousel logic, interval management, progress indicators  


## Features
- Automatic testimonial rotation with configurable timing
- User profile cards with images, names, and roles
- Smooth fade transitions between testimonials
- Manual navigation with previous/next arrows
- Visual progress bar showing time until next testimonial
- Responsive design with centered card layout


## Improvements Made
- **Timeout progress bar** – Added progress bar for each slide
- **Polished styling** – Refined overall design with shadows, borders, and modern themes
- **Navigation arrows** – Added manual skip buttons for user control


## What I Learned
- Interval-based content rotation
- Progress bar animations synchronized with timers
- Manual navigation that resets automatic intervals
- Array index management for circular navigation
- CSS transitions for smooth content changes
- Combining automatic and manual navigation controls
