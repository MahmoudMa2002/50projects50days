# Rotating Navigation - Metal Gear Solid Themed

An interactive rotating navigation interface with smooth page rotation and circular menu controls.


## Preview
![Project Screenshot](./Demo.gif)


## Info
**Tech:** HTML, CSS (Transforms, Transitions), JavaScript  
**Focus:** CSS animations, advanced positioning, event handling  


## Features
- Interactive page rotation that reveals hidden navigation menu
- Circular control interface with hamburger and close button states
- Staggered menu animation with sequential item reveals
- Custom Metal Gear Solid theming and content
- Smooth transitions and visual feedback


## Improvements Made
- **Enhanced visual design** – Applied Metal Gear Solid color scheme and typography unique to Metal Gear Solid
- **Content overhaul** – Added Metal Gear Solid universe content with fitting images instead of Lorem text


## What I Learned
- CSS transform origin for controlling rotation anchor points
- CSS transitions with timing control for smooth animations
- Event handling and class state toggling
- Advanced positioning with fixed/absolute for layered interfaces
- CSS pseudo-selectors for staggered animations
- Custom font integration with external font APIs
